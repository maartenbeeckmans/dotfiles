# dotfiles

Configurations:

- alacritty
- dunst
- i3
- kitty
- nitrogen
- polybar
- ranger
- redshift
- rofi
- vim
- zsh

## Usage

- For backing up the dotfiles, run the `backup-dotfiles.sh` script
- For restoring the dotfiles, run the `restore-dotfiles.sh` script

> The only dependency of this script is `rsync`
